# Documentation Directory

This directory contains the core package documentation for `kmds-featurization`.

## Purpose

These files are organized for compatibility with Open Knowledge Foundation / Frictionless-style packaging. The `datapackage.json` file provides metadata and a resource index for the Markdown documents in this directory.

## Usage

- `datapackage.json` describes the documents in this folder.
- Tools that support Frictionless data packages can read this directory as a package of documentation resources.
